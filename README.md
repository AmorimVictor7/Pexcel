Pexcel

O Pexcel é um sistema web desenvolvido para automatizar o cadastro de dados em planilhas, substituindo processos manuais por uma solução digital mais rápida, organizada e segura.

O sistema permite que usuários preencham um formulário online com campos dinâmicos que são carregados diretamente de uma planilha. Após o envio, os dados são registrados automaticamente em uma nova linha, evitando sobrescritas e reduzindo erros de digitação.

Este projeto foi criado com o objetivo de otimizar processos internos, melhorar a produtividade e facilitar o gerenciamento de informações.

 Funcionalidades
- Preenchimento de formulário online
- Carregamento automático de opções em campos select
- Envio de dados diretamente para planilhas
- Registro automático sem sobrescrever informações
- Interface simples e intuitiva
- Validação de campos antes do envio

Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Google Apps Script  
- Planilhas Google  
- Vercel (Deploy)

Objetivo do Projeto
Automatizar o processo de cadastro de informações que anteriormente era feito manualmente, reduzindo erros humanos, economizando tempo e garantindo maior organização dos dados.

Contexto
Este sistema foi desenvolvido como parte de um projeto prático voltado à melhoria de processos internos, demonstrando habilidades em desenvolvimento web, integração com APIs e automação de tarefas.
